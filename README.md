C# (ASP.NET Core), Angular, Kotlin – Projet full-stack académique : développement d’un écosystème e-commerce complet (site web + application mobile) basé sur une architecture 5 couches, API REST sécurisée, et tests unitaires.

- Backend : ASP.NET Core (architecture 5 couches : UI → Application → Domain → Infrastructure → Persistence), EF Core, migrations, DTOs, services, repository pattern
- Frontend web : Angular (routing, services, reactive forms, composants dynamiques)
- Application mobile : Kotlin / Android connectée au même backend REST (API partagée)
- Sécurité : JWT, validation des entrées, gestion des rôles (admin / utilisateur)
- Tests : tests unitaires (xUnit) sur la logique métier et les services
